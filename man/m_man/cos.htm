cos - Function

Trigonometric function


Description

Returns the cosine of an argument given in degrees.

Syntax

cos(angle)

FLOAT angle;


Principal parameters

angle - Angle in the range from -360 to +360 degrees


Optional parameters

None.


Return value

A FLOAT value in the range from -1.0 to 1.0
