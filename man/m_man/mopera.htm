Operators

Operators are things like plus and minus. Here is a list of the operators in MBS:

+ Addition
- Subtraction
* Multiplication
/ Division
** Raised to
<> Not equal
= Equal
< Less than
> Greater than
<= Less than or equal
>= Greater than or equal


For each datatype the following operators may be used:

INT All
FLOAT All
VECTOR + - * <> =
STRING + = < <= > >= <>
REF = <>
FILE None
