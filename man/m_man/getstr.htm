getstr - Function

Get parameter from GM


Description

Returns the value of a STRING parameter.

Syntax

getstr(part_id, parnum)

REF part_id;
INT parnum;


Principal parameters

part_id - Global identity of a part
parnum - Parameter number, first parameter = 1


Optional parameters

None.


Return value

A STRING parameter value.
