getref - Function

Get parameter from GM


Description

Returns the value of a REF parameter.

Syntax

getref(part_id, parnum)

REF part_id;
INT parnum;


Principal parameters

part_id - Global identity of a part
parnum - Parameter number, first parameter = 1


Optional parameters

None.


Return value

A REF parameter value.
