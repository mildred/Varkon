psh_men - Procedure

Pushes a menu


Description

Displays a menu in the menu window and pushes it to the menu stack.

Syntax

psh_men(number)

INT number;


Principal parameters

number - The number of the menu to display


Optional parameters

None.


Comments

Use pop_men() to remove a menu.
