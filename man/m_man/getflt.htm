getflt - Function

Get parameter from GM


Description

Returns the value of a FLOAT parameter.

Syntax

getflt(part_id, parnum)

REF part_id;
INT parnum;


Principal parameters

part_id - Global identity of a part
parnum - Parameter number, first parameter = 1


Optional parameters

None.


Return value

A FLOAT parameter value.
