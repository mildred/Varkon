    IGMAXID

    System variable
     

    Description

    IGMAXID is the maximum number of entities in a selection using the window method. If you specify a window area that contains more than IGMAXID entities only the first IGMAXID entities will be selected.

    The current value of IGMAXID is 5000 and can not be changed by the user.
