    f151 Create plot file

    Description

    Writes all or part of a model to file in VARKON PLT-format. PLT-format can be used for plotting or to read back again as a symbol.

    Available in

    2D and 3D.

    MBS Correspondence

    plot_view() and plot_win().

    Comments

    VARKON PLT-format conforms to the GKS Metafile format standardised by ISO.
