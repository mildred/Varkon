    f8 Create part in temporary mode

    Description

    Exactly the same function as f25-Create Part but does not add the corresponding part-statement to the active module. A part is created and stored in GM but only on a temporary basis until next time the active module is executed (f5).

    Available in

    2D and 3D generic mode.

    MBS Correspondence

    None.

    Comments

    To call from a menu use action code "r".
