VARKON Version 1.16 1998-11-03

f188 Set view scale

Description

Use this dialog window to alter the current scalefactor of the view in the graphics window. Select one of the buttons or enter a scalefactor in the text field. A scalefactor of 1.0 displays all geometry in actual size in millimeters. A scalefactor < 1 makes everything look smaller and a scalefactor > 1 makes everything look larger.

Available in

2D and 3D.

MBS Correspondence

scl_view() and rep_view()

Comments

None.
