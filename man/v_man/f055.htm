    f55 Create group

    Description

    Creates a group with name and one or more members. Maximum number of members in a group is 100 and the maximum number of groups that an entity can be member in is 3.

    Available in

    2D and 3D.

    MBS Correspondence

    group(id, name, ref1, ref2........);
