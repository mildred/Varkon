    f144 Write active module to file in MBS-format

    Description

    Creates the MBS-version of the currently active module and writes the result to a named file. This function is typically used to convert an interactively created module into a MBS-program.

    Available in

    2D and 3D generic mode.

    MBS Correspondence

    None.

    Comments

    MBS-files usually have names ending with ".MBS". You can use this function to create files with other names but VARKON only recognizes ".MBS" as a MBS-file.

    See also "Edit active module" (f216) and "Display active module in MBS format" (f146).
