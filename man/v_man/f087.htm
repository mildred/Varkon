VARKON Version 1.15 1997-10-16

    f87 Set active XFONT attribute to centerline

    Description

    Sets the currently active XFONT attribute to 2. All hatching created after setting XFONT to 2 is displayed using centerlines.

    Available in

    2D only.

    MBS Correspondence

    set(XFONT=2);

    Comments

    XFONT can be 0, 1 or 2. Default value for XFONT when a new job is started is 0.

    See also

    f85 , f86 , f88
