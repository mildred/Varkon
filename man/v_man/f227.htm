    f227 Set active TPMODE value

    Description

    Sets the currently active value of the TPMODE attribute. Texts created after setting TPMODE are displayed in accordance with the new value of TPMODE.

    TPMODE is used to control the display orientation of texts in 3D. A text with TPMODE = 0 is always displayed in the plane of the active view giving a 2D appearence. A text with TPMODE = 1 is displayed as a true 3D entity.

    You should use TPMODE = 0 if you want to add text to a model hat should be easy to read in any view. TPMODE = 1 should be used when you want true 3D behaviour. The text will then only be readable in appropriate views.

    Available in

    3D only.

    MBS Correspondence

    set(TPMODE=value);

    Comments

    TPMODE can have the value of 0 or 1. Default value for TPMODE when a new job is started is 0.
