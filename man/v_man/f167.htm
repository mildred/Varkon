VARKON Version 1.15 1997-10-16

    f167 Set active CDASHL value

    Description

    Sets the currently active value for the CDASHL attribute. Curves with CFONT = 1 or CFONT = 2 created after setting CDASHL are displayed with the new dashlength.

    Available in

    2D and 3D.

    MBS Correspondence

    set(CDASHL=value);

    Comments

    CDASHL can have any decimal or integer value > 0. Default value for CDASHL when a new job is started is 3.0.
