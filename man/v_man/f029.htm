    f29 Create 3D line tangent to 2 arcs

    Description

    Creates a line tangent to two different circular arcs. An integer alternative is used to distinguish between the 4 possible solutions.

    The new line will always be created in the XY-plane of the currently active coordinate system. If the old arcs are not in the XY-plane of the currently active coordinate system they will first be projected onto the XY-plane. The new line will then be created using the projected arcs.

    Available in

    3D.

    MBS Correspondence

    lin_tan2(id, ref1, ref2, alt);

    Comments

    None.
