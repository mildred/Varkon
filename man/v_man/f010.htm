    f10 Delete entity

    Description

    Used to delete a single entity or part entity. VARKON removes all data related to the indicated entity from the GM database. If the entity is a part this means removing all entities belonging to the part, its parameters and the part itself. In generic mode VARKON first checks if the entity is referenced and if not also removes the corresponding statement in the active module.

    If an entity is referenced VARKON will not let you delete the entity unless you are working in explicit mode. Instead VARKON will give you the option to blank the entity and thus remove it from your screen but not from the active module or the GM database.

    Available in

    2D and 3D generic and explicit mode.

    MBS Correspondence

    The macro statement del_mbs();

    Comments

    See also the del()-procedure in MBS
