    f78 Change a TSIZE attribute

    Description

    Used to edit the size of a text. TSIZE controls character height directly and character width indirectly by the TWIDTH factor.

    Available in

    2D only.

    MBS Correspondence

    None.

    See also

    f79, f89, f171, f226
