    f89 Change a TSLANT attribute

    Description

    Used to edit the character slant of a text. TSLANT is measured in % of 45 degrees. A value of 0 gives characters with no slant and positive values increases slant in the forward direction. Negative slant is also supported in which case caracters are backward slanted.

    Available in

    2D only.

    MBS Correspondence

    None.

    See also

    f78, f79, f171, f226
