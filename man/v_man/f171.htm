    f171 Change a TFONT attribute

    Description

    Used to select a different font for a text. Two fonts are supplied with the standard system. TFONT=0 (default) and TFONT=1. Additional fonts may be created by the user.

    Available in

    2D only.

    MBS Correspondence

    None.

    Comments

    Within a given font TSIZE, TWIDTH and TSLANT can be used to control the actual appearence. See below.

    See also
