VARKON Version 1.15 1997-10-16

    f210 Copy entities

    Description

    Select the entites you want to copy and enter a base position, a new position and the number of copies you want. VARKON copies the selected entites from the base position to the new position.

    This function is only available when VARKON is used in explicit mode.

    Available in

    2D explicit mode.

    MBS Correspondence

    None.

    Comments

    None.
