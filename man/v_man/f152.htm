    f152 Call macro module

    Description

    Asks for the name and parameters of a MACRO module and executes it. No result is stored in GM and no statement is added to the currently active module by f152.

    Available in

    2D and 3D.

    MBS Correspondence

    None.

    Comments

    See also f25 Create part and f8 Create part in temporery mode.
