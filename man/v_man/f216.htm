    f216 Edit active module in MBS format

    Description

    Starts the MBS-editor and displays the currently active module in MBS-format. Statements may then be added, edited or deleted using normal text editing commands.

    In VARKON for Windows the menubar of the editor includes a "compile" option that should be used before leaving the editor if any changes are made to the text. If changes are made VARKON automatically reexecutes the module when you exit the editor.

    In VARKON for UNIX resources varkon.mbsedit.autocompile and varkon.mbsedit.autoexec control weather compilation and reexecution should be done manually or automatically when leaving the editor.

    Available in

    2D and 3D generic mode.

    MBS Correspondence

    None.

    Comments

    Note that !comments entered into the active module are not saved by the compiler.

    In VARKON for UNIX you can select your favourite editor through the VARKON resourcefile and you can also pass additional parameters to the MBS compiler.
