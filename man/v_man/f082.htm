    f82 Set active DNDIG attribute

    Description

    Sets the currently active value for the number of digits attribute for dimensions.

    Available in

    2D.

    MBS Correspondence

    set(DNDIG=value);

    Comments

    Default value for DNDIG when a new job is started is 0.

    See also

    f80 , f81
