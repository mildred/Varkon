    PLYMXV

    System variable
     

    Description

    PLYMXV is the maximum number of vectors in a graphical object. A circle for example is displayed as a sequence of vectors (short straight lines). The number of vectors depends on the size of the circle and the current value of the curve accuracy factor set by the user. Othe objects that are displayed with many vectors are curves and surfaces, crosshatch, dashed or dotted lines/arcs/curves etc.

    PLYMXV is currently set to 20000 and can not be changed by the user. If by some reason a graphical object should need more than PLYMXV vectors to be displayed only the first PLYMXV vectors will be displayed.
