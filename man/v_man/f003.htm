VARKON Version 1.15 1997-10-16

    f3 Change parameter default value

    Description

    Changes the default value of a parameter belonging to the currently active module. VARKON asks for the name of the parameter and then prompts for the new value.

    Available in

    2D and 3D generic mode.

    MBS Correspondence

    Default value changed.

    Comments

    The active module is not automatically reevaluated (executed) by f3. Use f5 "Run active module" to see the effect of the changed parameter.
