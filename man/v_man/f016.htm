VARKON Version 1.15 1997-10-16

    f16 Set grid location

    Description

    Sets the origin of the current grid to the specified location.

    Available in

    2D.

    MBS Correspondence

    gridx_view() and gridy_view() for X and Y respectively.

    Comments

    See also f17 and f18 for setting the grid size or f114 and f115 to unblank or blank the grid.
