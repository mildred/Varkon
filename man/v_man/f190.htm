f190 Set view perspective

Description

If turned on, displays the model in the current view with true perspective projection. Increase the perspective distance to make the effect of perpective smaller. Decrease the distance to make the effect of perspective larger.

If turned off (default) standard parallell projection is used in this view.

Available in

3D only.

MBS Correspondence

perp_view()

Comments

The perpective distance must be larger than the size of the model.
