VARKON Version 1.15 1997-10-16

    f145 Create symb

    Description

    Reads a VARKON plot file (PLT-format) from disc and creates a part entity containing all the lines from the plot file. The name of the part is set to the name of the PLT file without the PLT extension.

    Available in

    2D.

    MBS Correspondence

    symb();

    Comments

    None.
