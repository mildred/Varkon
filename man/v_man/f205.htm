    f205 Edit arc start angle

    Description

    Edit function for circular acrs. Indicate an arc and enter a new start angle. Arcs run from a start angle to an end angle in anti clockwise direction.

    This function is only available when VARKON is used in explicit mode.

    Available in

    2D explicit mode.

    MBS Correspondence

    None.

    Comments

    None.
