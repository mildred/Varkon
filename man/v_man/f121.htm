    f121 Save result as

    Description

    Save current result to disc with a new name. No other files are saved. In generic mode result file names have extension RES and in explicit mode result file names have extensions RIT. VARKON adds this automatically.

    Available in

    2D and 3D.

    MBS Correspondence

    None.

    Comments

    None.

    See also

    f119, f126 and f201
