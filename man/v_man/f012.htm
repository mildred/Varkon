    f12 Change a LEVEL attribute

    Description

    Used to edit the LEVEL attribute of an entity. VARKON supports 2000 individual levels with numbers 0 throug 1999. Level 0 is default when a new job is started. All 2000 levels can be blanked or unblanked individually. Entities on a blank level are not displayed. Default for all levels is unblank meaning that all entities are displayed.

    Available in

    2D and 3D.

    MBS Correspondence

    None.

    Comments

    None.
