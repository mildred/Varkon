VARKON Version 1.15 1997-10-16

    f66 Set active AFONT attribute to solid

    Description

    Sets the currently active AFONT attribute to 0. All arcs created after setting AFONT to 0 are displayed as solid.

    Available in

    2D and 3D.

    MBS Correspondence

    set(AFONT=0);

    Comments

    AFONT can be 0, 1 or 2. Default value for AFONT when a new job is started is 0.

    See also

    f67 , f68 , f69
