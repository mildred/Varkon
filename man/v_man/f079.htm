    f79 Change a TWIDTH attribute

    Description

    Used to edit the width of a text. Actual width is computed by multiplying the text height with the TWIDTH factor divided by 100. A TWIDTH of 50 for example will create characters half as wide as they are high. Negative values of TWIDTH are also supported in which case characters are mirrored.

    Available in

    2D only.

    MBS Correspondence

    None.
