    f156 Create Chordlength spline without tangents

    Description

    Creates a Chordlength spline by interpolating 2 or more positions. No tangents need to be specified. For more details, see cur_spline() in the MBS documentation.

    Available in

    2D and 3D.

    MBS Correspondence

    cur_spline();

    Comments

    None.
