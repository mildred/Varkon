    f81 Set active DASIZ attribute

    Description

    Sets the currently active value for the dimension arrow size attribute.

    Available in

    2D.

    MBS Correspondence

    set(DASIZ=value);

    Comments

    DASIZ can have any decimal or integer value > 0. Default value for DASIZ when a new job is started is 5.0.

    See also

    f80 , f82
