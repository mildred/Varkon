VARKON Version 1.15 1997-10-16

    f163 Create cur_offs

    Description

    Creates a curve with constant offset to another curve. Which side of the old curve VARKON will use is determined by the side that was used to select the curve with the mouse. A positive value for offset will create a new curve on the side used for selection and a negative value for offset will create a new curve on the opposite side. In any case the new curve is always created in the XY-plane of the currently active coordinate system.

    Available in

    2D and 3D.

    MBS Correspondence

    cur_offs();

    Comments

    None.
