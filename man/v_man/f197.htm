f197 Manage levels

Description

This function blanks, unblanks or lists information about a one or more levels. Select the first level you want to manage and optionally how many additional levels (forward or backward) or the number of the last level and click the OK button.

Here is more information on levels.

Available in

2D and 3D.

MBS Correspondence

blank_lev(), unblank_lev() and get_lev()

Comments

None.
