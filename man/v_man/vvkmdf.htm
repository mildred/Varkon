VARKON Version 1.15 1997-10-16

    VARKON_MDF

    Environment variable


    Description

    Environment variable used by VARKON to find the system menu files. One of these is varkon.MDF wich is always loaded before any other menus are loaded each time the system is started. Other files are for examle v318.MDF which is the default menufile for new projects on a UNIX machine with X-Windows or v319.MDF which is the default menufile for new projects on a PC with Microsoft Windows 95 or NT.

    VARKON_MDF usually points to $VARKON_ROOT/mdf/swedish or $VARKON_ROOT/mdf/english depending on what language is requested.

    The UNIX version of VARKON also supports serial ASCII based Tektronix/VT300 compatible graphics terminals of different kind. For this purpose there are menu files in VARKON_MDF named v30.MDF through v317.MDF.

    Comments

    If VARKON_MDF is not defined a hard coded default will be used instead.

    If VARKON_MDF points to a non existent file an error is reported.

    The files in $VARKON_MDF should not be altered. Dont edit any files, add any files or remove any files. Project specific menu files should be placed where the application resides for example in $HOME/app/xxx/mdf if xxx is the name of the project. A project specific menu file may #include any file from $VARKON_MDF, usually v318.MDF or v319.MDF. The names of these files will not be changed in the future.
