    f80 Set active DTSIZ attribute

    Description

    Sets the currently active value for the dimension text size attribute.

    Available in

    2D.

    MBS Correspondence

    set(DTSIZ=value);

    Comments

    DTSIZ can have any decimal or integer value > 0. Default value for DTSIZ when a new job is started is 5.0.

    See also

    f81 , f82
