    f23 Create 3D offset line

    Description

    Creates a line parallell to another line at a given distance. If the old line is picked with the cursor on its left side the new line will be created on the left side. With the cursor on the right side of the line the new line will be created on the right side.

    The new line will always be created in the XY-plane of the currently active coordinate system. If the old line is not in the XY-plane of the currently active coordinate system the old line will first be projected onto the XY-plane. The new line will then be created at the specified distance from the projected copy of the old line.

    Available in

    3D.

    MBS Correspondence

    lin_offs(id, ref, distance);

    Comments

    The new line gets the same sense of direction as the old line.

    It is not an error to specify a zero distance. This can be used to "copy" a line from one coordinate system to another.
