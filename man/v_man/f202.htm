VARKON Version 1.15 1997-10-16

    f202 Drag entity

    Description

    Simple editing function used to move a single entity from one position to another. Indicate the entity and click the mouse in the position you want to move it. For lines, arcs and curves the closest end of the indicated entity is moved to the new position and for other entities their base position is moved.

    This function is only available when VARKON is used in explicit mode.

    Available in

    2D explicit mode.

    MBS Correspondence

    None.

    Comments

    None.
