    VARKON_PLT

    Environment variable


    Description

    Environment variable used by VARKON to locate configuration files for the plotting system. Some of these files may need editing to make plotting work properly, especially on UNIX systems or systems with more than one plotter. In order to keep these site specific changes when a new version of VARKON is installed it is wise to place the config. files in a directory outside $VARKON_ROOT. Use $HOME or some other directory not related to the VARKON system.

    Comments

    If VARKON_PLT is not defined a hard coded default will be used.

    If VARKON_PLT points to a non existent directory or if read/write permission to $VARKON_PLT is missing plotting will not work properly.
