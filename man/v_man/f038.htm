    f38 Create xhatch

    Description

    Creates a hatch pattern in an area bounded by lines, arcs and/or curves. xhatch requires at least one (outer) boundary but also accepts any number of inner boundaries (islands or holes). All boundaries must be closed ie. the entities that make up the boundary must be connected end to end. Boundaries may not cross or contact each other.

    Indicate the entities forming the boundary(ies) in any order and then enter a hatch line distance and angle.

    Available in

    2D.

    MBS Correspondence

    xhatch();

    Comments

    None.
