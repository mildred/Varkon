VARKON Version 1.15 1997-10-16

    f92 Create DXF file

    Description

    Writes all or part of a 2D model to file in DXF-format. VARKON entities are mapped to DXF entities as follows:

    • VARKON point to POINT
    • VARKON line to LINE
    • VARKON arc to ARC or CIRCLE
    • VARKON text to TEXT
    • All other VARKON entities to POLYLINE

    VARKON pen numbers are mapped directly to DXF pen numbers.

    VARKON levels (0-1999) are translated to DXF layer names by preceeding the level number with the text "LAY". VARKON level 15 will be written as "LAY15".

    VARKON linefonts are mapped to corresponding linetypes by preceeding the dashlength with the text "DSH" for font 1 and "CEN" for font2. A dashed linetype with 5 mm. dashlength will be written as "DSH5.0".

    Other systems may not know what "LAY15" or "DSH5.0" is but VARKON recognizes this convention and uses it if the file is imported back to VARKON again.

    Available in

    2D.

    MBS Correspondence

    plot_view() and plot_win().

    Comments

    A macro module in the VARKON library can be used to write 3D data to file in DXF-format.
