    f62 Create 2D circular arc with three positions

    Description

    Creates a circular arc specified by pos1, pos2 and pos3. The arc will turn in counter clocwise direction from pos1 through pos2 and end in pos3. If this is not possible VARKON automatically swaps pos1 with pos3 and lets the arc start in pos3, go through pos2 and end in pos1. In any case the arc turns in counter clockwise direction with a (arclength) parametrisation of t=0 where it starts and t=1 where it ends.

    Available in

    2D.

    MBS Correspondence

    arc_3pos(id, pos1, pos2, pos3);

    Comments

    If two positions coincide or if all three form a straight line an error is reported.
