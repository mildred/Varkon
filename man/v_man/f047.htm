    f47 Set active LFONT attribute to dashed

    Description

    Sets the currently active LFONT attribute to 1. All lines created after setting LFONT to 1 are displayed as dashed lines.

    Available in

    2D and 3D.

    MBS Correspondence

    set(LFONT=1);

    Comments

    LFONT can be 0, 1 or 2. Default value for LFONT when a new job is started is 0.

    See also

    f46 , f48 , f49
