    f217 Edit named module in MBS format

    Description

    Displays a list of all MBS-files in the mbs-directory of the current project and asks for one to be selected. VARKON then starts the MBS-editor and loads the selected file. Statements may be added, edited or deleted using normal text editing commands.

    In VARKON for Windows the menubar of the editor includes a "compile" option that should be used before leaving the editor if any changes are made to the text.

    In VARKON for UNIX the resource varkon.mbsedit.autocompile controls wether compilation should be done manually or automatically when leaving the editor.

    The module produced by the compiler (the MBO-file) is automatically copied to the lib-directory of the current project.

    Available in

    2D and 3D generic mode.

    MBS Correspondence

    None.

    Comments

    In VARKON for UNIX you can select your favourite editor through the VARKON resourcefile and you can also pass additional parameters to the MBS compiler.
