    f15 Delete last entity

    Description

    Deletes the last statement of the currently active module.

    Warning: Any corresponding graphics or information in GM is NOT deleted  !

    If f15 is used to delete a geometric statement the user is responsible of also deleting the corresponding entity in GM. An safe way to do this is to reevaluate the entire module. (Run actve module, f5)

    The main purpose of f15 is to make it possible to remove non geometric statements.

    Available in

    2D and 3D generic mode.

    MBS Correspondence

    None.

    Comments

    Be careful when using this function. Always reevaluate the module after using it.
