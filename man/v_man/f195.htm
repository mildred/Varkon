VARKON Version 1.16 1998-11-03

f195 Select view

Description

Select one of the currently defined views to see the model from a different direction or in different size.

Available in

2D and 3D.

MBS Correspondence

act_view() and rep_view()

Comments

Use the graphics/view/create-menu to create up to maximum 20 different views.
