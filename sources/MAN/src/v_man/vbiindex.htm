    Interactive VARKON functions

    The VARKON system includes more than 200 interactive functions written in C and numbered indivudally as f1, f2, f3 etc. Another 35 functions are written in MBS. You can acess these functions through menus, toolbars and ProgrammableFunction keys (PF-keys). The standard user interface ($VARKON_MDF/varkon.MDF) includes a setup for this.

    Most functions have doc-pages of their own but not all. Please have patience ! We are working on it !
