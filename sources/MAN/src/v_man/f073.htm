VARKON Version 1.15 1997-10-16

    f73 Change a *DASHL attribute

    Description

    Used to edit the dash length attribute of an entity. Pick the entity with the mouse and enter a new dashlength.

    • LDASHL for lines
    • ADASHL for arcs
    • CDASHL for curves
    • XDASHL for hatching

    Available in

    2D and 3D.

    MBS Correspondence

    None.
