    f88 Set active XDASHL value

    Description

    Sets the currently active value for the XDASHL attribute. Hatches with XFONT = 1 or XFONT = 2 created after setting XDASHL are displayed with the new dashlength.

    Available in

    2D only.

    MBS Correspondence

    set(XDASHL=value);

    Comments

    XDASHL can have any decimal or integer value > 0. Default value for XDASHL when a new job is started is 3.0.

    See also
