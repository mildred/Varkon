    f9 Create 2D coordinate system

    Description

    Creates a coordinate system with name, position and rotation around the Z-axis.

    Available in

    2D.

    MBS Correspondence

    csys_1p(id, name, pos, 0, 0, rot_z);

    Comments

    See also f106 and f107 for activation and deactivation of coordinate systems.
