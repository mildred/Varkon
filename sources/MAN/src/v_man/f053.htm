    f53 Set active TSLANT value

    Description

    Sets the currently active value of the TSLANT attribute. Texts created after setting TSLANT are displayed with the new text slant. TSLANT is measured in % of 45 degrees.

    Available in

    2D only.

    MBS Correspondence

    set(TSLANT=value);

    Comments

    TSLANT can have any positive or negative decimal or integer value. Default value for TSLANT when a new job is started is 15.0 %.
