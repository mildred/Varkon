    f186 Set active WIDTH value

    Description

    Sets the currently active value for the WIDTH attribute. Lines, arcs and curves created after setting WIDTH are displayed with the new linewidth.

    Available in

    2D and 3D.

    MBS Correspondence

    set(WIDTH=value);

    Comments

    WIDTH may have any decimal or integer value >= 0. Default value when a new job is started is 0.0.
