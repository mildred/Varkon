    f199 Display shaded image (Gourad)

    Description

    Exactly the same function as f198 Display shaded image (Flat) with the difference that this function uses the smoother Gourad shading model of OpenGL instead of the flat model. Gourad shading produces better images for the same number of facets if the light distribution is not the same everywhere in the scene. This is the case for example when spotlights are used.

    Available in

    3D.

    MBS Correspondence

    shade_view();

    Comments

    There are currently no interactive functions to create material or light sources but MBS includes routines for this.
