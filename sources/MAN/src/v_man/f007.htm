    f7 Edit part parameter

    Description

    This function is the old Vt100/Tektronix based version of function f170 for Microsoft Windows and UNIX X-windows used to edit one or more parameters in a part.

    Available in

    2D and 3D.

    MBS Correspondence

    The updpp_mbs() macro statement.

    Comments

    See f170 for details.
