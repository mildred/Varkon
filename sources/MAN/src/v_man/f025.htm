    f25 Create part

    Description

    Creates a part by asking for the name of its creating module and its parameters. The module is executed and the resulting part with its entities and parameters is stored in GM. In generic mode a corresponding part()-statement is also generated and added to the end of the currently active module.

    Available in

    2D and 3D.

    MBS Correspondence

    The add_mbs() macro can be used to create part statements.

    Comments

    See also f8 Create part in temporary mode and f152 Call macro module.
