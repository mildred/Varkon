    f39 Create 2D line perpendicular to other line

    Description

    Creates a line with specified start position and length and also perpendicular to another line. If the old line is picked with the cursor on its left side the new line will go in that direction. If the old line is picked with the cursor on its right side the new line will go in the opposite direction.

    Available in

    2D.

    MBS Correspondence

    lin_perp(id, pos, ref, length);

    Comments

    length must be greater than zero.
