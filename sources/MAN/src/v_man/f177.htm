    f177 Activate basic coordinate system

    Description

    Activates VARKON's basic coordinate system. Effectively the same as activating the global system of the active (topmost) module.

    Available in

    2D and 3D.

    MBS Correspondence

    mode_basic();

    Comments

    See also f106 and f107.
