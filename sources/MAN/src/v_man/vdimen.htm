    Dimensioning

    VARKON includes four entities for dimensioning. They are....

    ldim(id,p1,p2,p3,horisontal/vertical/parallel); ! Linear

    cdim(id,arc_id,p1,horisontal/vertical/parallel); ! Circular

    rdim(id,arc_id,p1,p2); ! Radius

    adim(id,lin1_id,lin2_id,p1,alternative); ! Angular

    The style is European and suited for mechanical applications. Dimensions can only be used in a DRAWING MODULE.

    Some features are controlled using attributes. DTSIZ controls textsize in dimensions while DNDIG controls the number of decimal places. DASIZ controls the size of arrows and DAUTO toggles automatic dimensioning on/off.

    Many VARKON users add new dimensioning styles and drafting symbols to their systems when they need them. MBS is well suited for parametric objects like this and it is usually very easy to build a library of your own for a certain application.
