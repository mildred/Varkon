    f109 Update part

    Description

    Used to reevaluate a single module. Indicate wich part you want to update by picking any one of the entities that belong to the part. VARKON then deletes the old part from GM and creates a new by calling the corresponding module again with the same parameters. f109 can be useful if information that the part depends on has changed and you dont want to reevaluate the entire model.

    Available in

    2D and 3D.

    MBS Correspondence

    MBS-macro updpp_mbs()

    Comments

    None.

    See also

    f5 , f170
