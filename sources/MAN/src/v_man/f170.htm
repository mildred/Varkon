    f170 Edit part parameters

    Description

    Used to edit one or more parameters in a part. Indicate wich part you want to edit by picking any one of the entities that belong to the part. VARKON then presents the edit window with all parameters of the indicated part, their names, values and promts. You can edit one parameter or many in any order you like. VECTOR-parameters may be edited using the position menu by clicking the "down arrow" next to the parameter. REF-parameters may be edited by mouse picking in the same way.

    When all parameters are edited click the OK-button. VARKON then automatically reevaluates the part with the new parameters. If the part is referenced (has other entities associated to it) VARKON asks you to reevaluate the entire active module. You may omit this if you want to (to save time) but referencing entities will then not be updated. It is always best to reevaluate the entire module if the edited part is referenced.

    If the edited part is not referenced VARKON only reevaluates the edited part.

    Available in

    2D and 3D.

    MBS Correspondence

    MBS-macro updpp_mbs()

    Comments

    In explicit mode there is no active module and VARKON only reevaluates the edited part even if it is referenced.
