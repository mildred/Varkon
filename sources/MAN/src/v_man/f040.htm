    f40 Create 2D line

    Description

    Creates a line by joining two positions. The line will begin in the first position and end in the second position with a linear parametrisation of t=0 in the start position to t=1 in the end position.

    Available in

    2D.

    MBS Correspondence

    lin_free(id, pos1, pos2);

    Comments

    pos1 must not be equal to pos2.
