VARKON Version 1.15 1997-10-16

    f164 Set active CFONT attribute to solid

    Description

    Sets the currently active CFONT attribute to 0. All curves created after setting CFONT to 0 are displayed as solid.

    Available in

    2D and 3D.

    MBS Correspondence

    set(CFONT=0);

    Comments

    CFONT can be 0, 1 or 2. Default value for CFONT when a new job is started is 0.
