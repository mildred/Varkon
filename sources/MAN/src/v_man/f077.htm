    f77 Change a *FONT attribute to centerline

    Description

    Used to set the font attribute of an entity to solid.

    • LFONT=2 for lines
    • AFONT=2 for arcs
    • CFONT=2 for curves
    • XFONT=2 for hatching

    Available in

    2D and 3D.

    MBS Correspondence

    None.
