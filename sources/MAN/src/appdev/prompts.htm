Prompt strings

The Create part function which is used to interactively call a module understands a wide variety of different ways to declare parameters with prompt strings. You can use this mechanism to finetune the behaviour of your modules when they are used interactively.

For more information, see the Parameter delaration section of the MBS programmers manual.
