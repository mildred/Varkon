ln - Function

Description

Returns the natural logarithm of an argument.

Syntax

  y:=ln(x);

  float y;
  float x;

Principal parameters

x - A value > 0.0

Optional parameters

None.

Return value

A FLOAT value equal to the natural logarithm of x.
