log - Function

Description

Returns the base-10 logarithm of an argument.

Syntax

  y:=log(x);

  float y;
  float x;

Principal parameters

x - A value > 0.0

Optional parameters

None.

Return value

A FLOAT value equal to the base-10 logarithm of x.
