outlin - Procedure

Write end-of-line to a file


Description

Writes an end-of-line character to a file.

Syntax

outlin(f)

FILE f;


Principal parameters

f - The name of a filevariable accociated with a file open for write, append or update (W, A or U)


Optional parameters

None.
