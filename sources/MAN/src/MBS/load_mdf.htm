load_mdf - Procedure

Loads MDF-file


Description

Reads an MDF-file and merges all menus, t-strings, accelerators etc. defined with those defined earlier.

Syntax

load_mdf(name)

STRING name*132;


Principal parameters

name - The name of the file to load, path may be included


Optional parameters

None.
