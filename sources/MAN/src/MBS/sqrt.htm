sqrt - Function

Description

Returns the square root of an argument.

Syntax

  y:=sqrt(x);

  float y;
  float x;

Principal parameters

x - A value greater than or equal to zero.

Optional parameters

None.

Return value

A FLOAT value equal to the square root of x.
