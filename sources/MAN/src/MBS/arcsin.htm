arcsin - Function

Trigonometric function


Description

Returns the arc sine of an argument as an angle measured in degrees.

Syntax

arcsin(x)

FLOAT x;


Principal parameters

x - Value in the range from -1.0 to 1.0


Optional parameters

None.


Return value

A FLOAT value in the range of -90 to +90 degrees
