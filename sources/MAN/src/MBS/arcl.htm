arcl - Function

Description

Returns the true length of a line, arc or curve.

Syntax

  l:=arcl(id);

  float l;
  ref   id;

Principal parameters

id - Identity of line, arc or curve.

Optional parameters

None.

Return value

A FLOAT value equal to the arclength of the referenced entity.
