pca - Geometric procedure

Description

Low level routine to create a plane by specifying set of points stored into a one dimension matrix. The result is 4x4 matrix represents the coordinate system. The XY plane of the coordinate system is the fitted plane to set of points.

Syntax

pca(id,npts,pts,tmat:attributes);
REF id;
INTEGER npts;
VECTOR pts(npts);
FLOAT tmat(4,4);

Principal parameters

id - Local identity
npts - Number of points
pts - Array of points
tmat - 4 by 4 transformation matrix

Optional parameters

None.

Attributes

PEN - Pen or colour number
LEVEL - Level number
BLANK - Visible(0) or invisible(1)

Comments

To visualize or use 4x4 matrix that represent the coordinate system use csys_usdef() procedure see below.

See also

csys_usrdef()
