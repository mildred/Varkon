unblank - Procedure

Description

Sets the BLANK attribute of an entity to 0. If not on a blanked level the entity will then be visible.

Syntax

  unblank(id);

  ref id;

Principal parameters

id - The identity of the entity to unblank.

Optional parameters

None.

Comments

See also: blank
