cos - Function

Description

Returns the cosine of an argument given in degrees.

Syntax

  x:=cos(angle);

  float x;
  float angle;

Principal parameters

angle - Angle in the range from -360 to +360 degree

Optional parameters

None.

Return value

A FLOAT value in the range from -1.0 to +1.0
