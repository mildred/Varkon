get_color - Procedure

Description

Returns the RGB color value for a specific pen (0-255).

Syntax

  get_color(pen, red, green, blue);

  int pen;
  int red;
  int green;
  int blue;

Principal parameters

pen - Pen number. 0 - 255
red - Red intensity, 0 - 255
green - Green intensity, 0 - 255
blue - Blue intensity, 0 - 255

Optional parameters

None.

Comments

See also cre_color
