vecl - Function

Geometrical function


Description

Returns the length of a vector.

Syntax

vecl(p)

VECTOR p;


Principal parameters

p - Any vector


Optional parameters

None.


Return value

A FLOAT equal to the length of the vector = sqrt(p.x**2 + p.y**2 + p.z**2)

Example

vecl(vec(100,100)) = 141.4......
