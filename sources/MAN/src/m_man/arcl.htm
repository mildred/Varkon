arcl - Function

Geometrical funktion


Description

Returns the true length of a line, arc or curve.

Syntax

arcl(ref)

REF ref;


Principal parameters

ref - Identity of line, arc or curve


Optional parameters

None.


Return value

A FLOAT value equal to the arclength of the entity referenced.
