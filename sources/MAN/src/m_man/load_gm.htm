load_gm - Procedure

Loads a new GM


Description

Replaces the current GM with the contents of a named resultfile.

Syntax

load_gm(resfile)

STRING resfile*132;

Principal parameters

resfile. - The name of a RES- or RIT-file to load. A path may be included.


Optional parameters

None.
