arctan - Function

Trigonometric function


Description

Returns the arc tangent of an argument as an angle measured in degrees.

Syntax

arctan(x)

FLOAT x;


Principal parameters

x -  Any value


Optional parameters

None.


Return value

A FLOAT value in the range of -90 to +90 degrees.
