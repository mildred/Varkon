grid_view - Procedure

Turns grid on/off


Description

Shows or hides the graphics grid on the display.

Syntax

grid_view(show)

INT show;


Principal parameters

show - 1 = Show grid, 0 = Hide grid


Optional parameters

None.


Comments

It is possible to use the grid even when it is not displayed.
