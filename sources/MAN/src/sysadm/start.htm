Start Varkon

Varkon can only be started by executing a Varkon start script. By default Varkon comes with an english start script located in $VARKON_ROOT/com/english. To start Varkon, run the Varkon start script. Varkon will then display the following dialog:

If this is the first time you start Varkon, you can now continue with the The 6 min tutorial for first time users.

More information about start scripts is available in the Start script section and also the Application developers manual.

If Varkon does not start when you execute the start script, check $VARKON_ROOT for a file named varkon.LOG. This file shows what happens during startup and may be a help to localize the problem.
