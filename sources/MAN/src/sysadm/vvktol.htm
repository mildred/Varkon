VARKON Version 1.15 1997-10-16

    VARKON_TOL

    Environment variable


    Description

    Environment variable used by VARKON to locate tolerance files. Version 1.14 of VARKON includes only one tolerance file named SURPAC.TOL.

    SURPAC.TOL lists values of different tolerances to be used for numerical calculations related to sculptured surfaces.


    Comments

    If VARKON_TOL is not defined a hard coded default will be used.

    If VARKON_TOL points to a non existent directory or if read permission to $VARKON_TOL is missing or if any of the files in $VARKON_TOL are missing no error is reported and default values for tolerances are used instead.
