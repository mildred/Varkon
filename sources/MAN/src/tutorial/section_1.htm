Drawings and modules

Varkon is used to create drawings. You can do this two ways:

  1. Directly. This is called explicit modeling. The result will be a RES-file (drawing).

  2. Indirectly, by first creating a module and then let the module create the drawing. This is called generic modeling. The result will be a MBO-file (module) and a RES-file (drawing).

The first method is the easiest to learn. The second method takes a little longer, but once there you will be able to do much more. This tutorial will show you how to use Varkon in generic mode. Doing so, you will also learn a lot about explicit modeling.
