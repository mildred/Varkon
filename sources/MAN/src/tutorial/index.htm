A 6 min tutorial for first time users

Welcome to Varkon. This tutorial will explain to you the difference between modules and RES-files so that you can start your first job. It will also show you how to navigate in menus, how to use the mouse, how to get help and how to create, edit and save a simple model. There are 6 sections and each takes a minute to complete. Start with section 1.

  1. Modules and RES-files
  2. Start your first job
  3. Create some entities
  4. Make some changes
  5. Play with MBS
  6. Save and exit

More information is available on the Varkon documentation pages and the Varkon project site on SourceForge. You may also want to join the Varkon mailing list to discuss specific topics with the Varkon community.
