    f43 Create 2D angular line

    Description

    Creates a line with specified start position, length and angle. Positive angles are measured as anti clockwise rotation relative to the X-axis of the currently active coordinate system.

    Available in

    2D.

    MBS Correspondence

    lin_ang(id, pos, angle, length);

    Comments

    length must be greater than zero.

    Angles should be in the range of -360 to +360.
