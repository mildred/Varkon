    f44 Create 2D line tangent to arc or curve

    Description

    Creates a line with specified start position ending in the position of tangency with a specified arc or curve. An integer alternative may be specified to select the right end position when there is more than one position of tangency available.

    Available in

    2D.

    MBS Correspondence

    lin_tan1(id, pos, ref, alt);

    Comments

    Tangency positions are enumerated along the arc or curve starting with one for the first possible position two for the second and so on. Negative values or zero may not be used.
