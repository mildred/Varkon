VARKON Version 1.15 1997-10-16

    f146 Display active module in MBS-format

    Description

    Creates the MBS-version of the currently active module and displays the result in a VARKON listwindow.

    Available in

    2D and 3D generic mode.

    MBS Correspondence

    None.

    Comments

    See also "Edit active module" (f216) and "Write active module to file in MBS format" (f144).
