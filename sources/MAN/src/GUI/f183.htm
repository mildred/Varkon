    f183 Create cur_int

    Description

    Creates a planar curve by intersecting a surface with a b_plane or the XY-plane of a coordinate system. If the intersect results in more than one curve (branch) VARKON enumerates them from 1 and up. If only one branch exists it is always branch number 1.

    Available in

    3D.

    MBS Correspondence

    cur_int();

    Comments

    None.
