    f122 Exit, save first

    Description

    Exit VARKON and save all job data to disc. In generic mode this means creating a JOB-file, MBO-file and RES-file. In explicit mode a JOB- and RIT-file is created. Any existing files are overwritten without notice.

    Available in

    2D and 3D.

    MBS Correspondence

    None.

    Comments

    None.
