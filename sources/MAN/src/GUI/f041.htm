    f41 Create projected 3D line

    Description

    Creates a line by joining two positions in 3D space and then projecting the result onto the XY-plane of the currently active coordinatesystem.


    Available in

    3D.


    MBS Correspondence

    lin_proj(id, pos1, pos2);
