VARKON Version 1.15 1997-10-16

    f46 Set active LFONT attribute to solid

    Description

    Sets the currently active LFONT attribute to 0. All lines created after setting LFONT to 0 are displayed as solid.

    Available in

    2D and 3D.

    MBS Correspondence

    set(LFONT=0);

    Comments

    LFONT can be 0, 1 or 2. Default value for LFONT when a new job is started is 0.

    See also

    f47 , f48 , f49
