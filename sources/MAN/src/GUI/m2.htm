The main menu

Select an alternative from the menu with the left mouse button or enter the first character(s) of the alternative from the keyboard. A sub menu will then be presented and you can make a new selection and so on...

To return from a sub menu one step back to the previous menu, click the right mouse button or press RETURN on the keyboard. To return directly back to the main menu use the middle mouse button or TAB on the keyboard.

Menus are defined by the text file defined by the value of the environment variable VARKON_MDF in the Varkon start script. See the menufile section in the docs for more information about menufiles.

To learn more about Varkon, see learn more.
