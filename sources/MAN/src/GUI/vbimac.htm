    Interactive MBS-functions

    VARKON also includes a number of functions written in MBS. For the end user of a VARKON application it is probably of no interest to know weather a function is written in C or MBS but for those who develop applications it can be important since the calling convention is different. Here are the major modules in the VARKON system library $VARKON_LIB.
     
    Name Description
    p_plott Setup and start a plot on a plotter
    p_autop Autoplot
    p_pwin Plot window
    ed_hk Edit entity
    ma_texth Create text right justified
    ma_textc Create text centered
    ma_text2 Create text between 2 positions
    dxf_2D Reads a 2D DXF file
    dxf_3D Reads a 3D DXF file
    stl_txt Reads text formatted STL file
    stl_bin Reads binary formatted STL file
    stl_out Creates STL file
    l_formel Evaluate mathematical expression
    l_anhk Analyse single entity
    l_anavst Analyse distance
    l_anvink Analyse angle
    l_area Analyse 2D area
    l_cgrav Analyse 2D center of gravity
    l_sysdat Lists various system info
    l_stjobb List current jobstatus
    l_stgrafv List current graphics parameters
    l_stattr List currently active attributes
    l_anaprt Analyse single part
    l_akedja Analyse call chain
    l_annapa List members in part
    l_lisp List parts
    l_anastr Create structure tree
    l_ligm List entities
    l_annagr List members in group
    l_anligr List groups
    l_anlgd Analyse curve length
    l_antang Analyse tangent
    l_annorm Analyse normal
    l_anbino Analyse binormal
    l_ankrok Analyse curvature
    l_invon Analyse inverse position
    l_invtang Analyse inverse tangent
    l_invcur Analyse inverse curvature
    l_inval Analyse inverse arc length
