Buttons

Text buttons

A text button displays a text. The texts displayed in menus are defined by the current menu file. The texts and tooltips displayed in most other buttons are defined in the current ini file.

Icons

Icons are buttons with a picture defined by an icon file in the VARKON_ICO directory. See the Varkon start script for more info.

Toolbars in graphical windows

Graphical windows have toolbars composed of icons. You can create your own toolbars (composed of text buttons and/or icons) in graphical windows by creating your own ini file.

Buttons and icons in MBS

You can also create your own windows with buttons and icons with MBS.
