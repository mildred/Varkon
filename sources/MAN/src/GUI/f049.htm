VARKON Version 1.15 1997-10-16

    f49 Set active LDASHL value

    Description

    Sets the currently active value for the LDASHL attribute. Lines with LFONT = 1 or LFONT = 2 created after setting LDASHL are displayed with the new dashlength.

    Available in

    2D and 3D.

    MBS Correspondence

    set(LDASHL=value);

    Comments

    LDASHL can have any decimal or integer value > 0. Default value for LDASHL when a new job is started is 3.0.
