VARKON Version 1.15 1997-10-16

    f120 Open new job, save nothing

    Description

    Leaves the current job wihout saving any data to disc and opens a new job.

    Available in

    2D and 3D.

    MBS Correspondence

    None.

    Comments

    In generic mode no result is written to disc. If a RES-file existed when the job was started VARKON removes it.
