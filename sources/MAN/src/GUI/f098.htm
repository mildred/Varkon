VARKON Version 1.15 1997-10-16

    f98 Set active curve accuracy

    Description

    Sets the current value used by VARKON to determine the number of straight line segments (vectors) or plane facets to be used in the display of curves and surfaces. A higher value gives a smoother image but is more CPU intensive and may slow down the system.

    Available in

    2D and 3D.

    MBS Correspondence

    cacc_view(value);

    Comments

    Curve accuracy can be set to any value in the range 0.01 to 100.0. Default value when a new job is started is 1.0.

    During the creation of a plot file VARKON always uses a curve accuracy which is twice as high as the current factor.
