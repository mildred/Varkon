    f2 Create a parameter

    Description

    Adds a new parameter to the currently active module. Parameters have name, type, prompt and default value.

    The name may consist of up to 31 letters and digits and the underline_character but must start with a letter. Upper or lower case is not significant. "Diameter" is the same as "DIAMETER" or "diameter".

    Type is chosen from a menu of INT, FLOAT, STRING or VECTOR.

    Prompt is an arbitrary string describing what the parameter does. If the currently active module later will be called interactively the prompt string will be presented along with the default value.

    Available in

    2D and 3D generic mode.

    MBS Correspondence

    type name:=value >"prompt"

    Comments

    Once added there is currently no dedicated interactive function to delete a parameter. Use f216 "Edit active module" if you need to change or delete a parameter.

    f3 "Change parameter value" can be used to change the default value.

    See the MBS Reference manual for more information on parameters.
