    f21 Create projected 3D point

    Description

    Creates a 3D point entity. Any X-, Y- and Z-coordinate may be specified but only the X- and Y-coordinates are used. The Z-coordinate is automatically set to zero. The resulting point will always be located in the XY-plane of the active coordinate system.

    Available in

    3D.

    MBS Correspondence

    poi_proj(id, pos);

    Comments

    See also f19 and f20 for other ways to create a point.
