    f198 Display shaded image (Flat)

    Description

    This is the main rendering function in VARKON. Produces a colored and shaded image of all visible surfaces of the current view taking any material defined or lightsources into account. f198 uses the flat shading model in OpenGL which produces the quickest possible result. B_planes are rendered as they are but surfaces are facetted (tesselated) by OpenGL before rendering. The quality of the tesselation is controlled by the value of VARKON:s current curve accuracy factor. A higer value (max=100) ubdivides surfaces into more facets and is more CPU intensive. A lower value (min=0.1) produces larger facets in a shorter time.

    Available in

    3D.

    MBS Correspondence

    shade_view();

    Comments

    There are currently no interactive functions to create material or light sources but MBS includes routines for this.
