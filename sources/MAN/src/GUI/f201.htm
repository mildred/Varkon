    f201 Save job as

    Description

    Saves current JOB-file, MBO-file (in generic mode) and RES/RIT-file to disc with a new name. This is a convinience function to save a complete job with a new name.

    Available in

    2D and 3D.

    MBS Correspondence

    None.

    Comments

    None.

    See also

    f119, f126 and f121
