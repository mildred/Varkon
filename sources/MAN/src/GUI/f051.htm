    f51 Set active TSIZE value

    Description

    Sets the currently active value of the TSIZE attribute. Texts created after setting TSIZE are displayed with the new text size. TSIZE sets the size of texts in absolute units. Capital "A" is used as a reference, ie. a TSIZE of 10 will make capital "A" 10 units high. Letters like "ÅÄÖ" will be higher and letters like "gjp" will reach under the baseline.

    Available in

    2D only.

    MBS Correspondence

    set(TSIZE=value);

    Comments

    TSIZE can have any decimal or integer value > 0. Default value for TSIZE when a new job is started is 5.0.
