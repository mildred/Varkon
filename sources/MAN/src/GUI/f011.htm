    f11 Change a PEN attribute

    Description

    Used to edit the PEN attribute of an entity. PEN attributes are used to control the color of an entity when displayed on the screen. PEN attributes can also be used to control linewidth on plotters and printers.

    • Pen 0 = The background color of the screen (usually white)
    • Pen 1 = The foreground color (usually black)
    • Pen 2 = Red
    • Pen 3 = Green
    • Pen 4 = Blue
    • And so on, see the "varkon_color" resources in your ini-file.

    Available in

    2D and 3D.
