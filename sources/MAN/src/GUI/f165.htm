VARKON Version 1.15 1997-10-16

    f165 Set active CFONT attribute to dashed

    Description

    Sets the currently active CFONT attribute to 1. All curves created after setting CFONT to 1 are displayed as dashed curves.

    Available in

    2D and 3D.

    MBS Correspondence

    set(CFONT=1);

    Comments

    CFONT can be 0, 1 or 2. Default value for CFONT when a new job is started is 0.
