    f90 Set active LEVEL attribute

    Description

    Sets the currently active value of the LEVEL attribute. All entities created after setting LEVEL are created with the new value (on the new level).

    Available in

    2D and 3D.

    MBS Correspondence

    set(LEVEL=value);

    Comments

    LEVEL can have any integer value in the range from 0 to 1999. 0 and 1999 included.
