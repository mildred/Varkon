    f70 Create Ferguson spline with tangents

    Description

    Creates a Ferguson spline curve that interpolates two or more positions and tangents. For more details, see cur_spline() in the MBS documentation.

    Available in

    2D and 3D.

    MBS Correspondence

    cur_spline();

    Comments

    None.
