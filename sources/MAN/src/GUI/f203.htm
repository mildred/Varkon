VARKON Version 1.15 1997-10-16

    f203 Rotate entities

    Description

    Select the entites you want to rotate and enter a position and angle. VARKON rotates the selected entites with the position as center in anticlockwise direction the number of degrees you specified. A negative angle rotates in clockwise direction.

    This function is only available when VARKON is used in explicit mode.

    Available in

    2D explicit mode.

    MBS Correspondence

    None.

    Comments

    None.
