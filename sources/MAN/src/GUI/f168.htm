    f168 MBS-Edit single entity

    Description

    Displays the MBS statement that defines the specified entity and lets you edit the statement using the keyboard. If the entity defined by the statement is not referenced by any other entity VARKON uses incremental mode to reevaluate the module (quicker) otherwise the module is evaluated in normal mode (slower). If an error occurs during reevaluation VARKON automatically reinstalls the original definition.

    Available in

    2D and 3D generic mode.

    MBS Correspondence

    None.

    Comments

    See also "Edit active module" (f216).

    There is currently a limit of the maximum number of characters in the MBS ststement that f7 can handle equal to 132 characters. Very long definitions like a cur_spline with many defining positions and tangents may not be possible to edit using this method.
