    f50 Create text

    Description

    Creates a text. A start position is given and an angle. The start position is the lower left corner of the first character in the string. Angles are measured from the X-axis of the currently active coordinatesystem and positive in anti clockwise direction. Maximum length of the string is 132 characters.

    Available in

    2D and 3D.

    MBS Correspondence

    text(id, pos, angle, string);

    Comments

    The TPMODE attribute can be used in 3D to control the display orientation of a text.
