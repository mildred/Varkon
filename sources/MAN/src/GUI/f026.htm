    f26 Create 3D B_plane

    Description

    Creates a triangular or four-sided bounded plane (facet) in 3D-space by defining four 3D positions. If position2 = position3 the plane becomes a triangle. Otherwise the plane is bounded by four straight lines. A b_plane has two sides, outside and inside. The outside is defined by following the defining positions in anti clockwise order. Viewed from the outside a b_plane is solid and hides everything behind it if displayed using the "hide" or "shade" function. Viewed from the inside a b_plane is transparent.

    The default graphical representation of a b_plane is wireframe.

    Available in

    3D.

    MBS Correspondence

    b_plane(id, p1 ,p2 ,p3 ,p4);

    Comments

    B_planes must be convex and all four positions must be coplanar. See also the sur_approx routine which can approximate a sculptured surface with facets (b_planes).
