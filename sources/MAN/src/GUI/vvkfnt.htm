VARKON Version 1.15 1997-10-16

    VARKON_FNT

    Environment variable


    Description

    Environment variable used by VARKON to locate files with fonts for vector based graphical text (the TEXT statement in MBS). Version 1.14 of VARKON includes two fonts named font 0 and font 1. Additional fonts can be defined by the user. Font 0 is the system default and its appearence is hardcoded within the system. Font 1 is defined in a fontfile named 1.FNT residing in $VARKON_FNT. 1.FNT is not loaded until it is used for the first time but then it stays loaded in memory until VARKON terminates.

    If a user defined font is created it should have a name beginning at 10.FNT and higer up to 20.FNT and reside in $VARKON_FNT.

    Comments

    If VARKON_FNT is not defined a hard coded default will be used.

    If VARKON_FNT points to a non existent directory or if read permission to $VARKON_FNT is missing no loadable fonts will be available and the system default (font 0) will be used instead.
