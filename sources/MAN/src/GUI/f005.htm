    f5 Run active module

    Description

    Reexecutes the currently active module from start to end. f5 first erases all information in GM and resets GM to its initial state as if a completely new job was started.

    Available in

    2D and 3D generic mode.

    MBS Correspondence

    The macro statement run_mbs()

    Comments

    None.

    See also

    f109 , f170
