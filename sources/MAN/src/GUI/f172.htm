    f172 Set active TFONT value

    Description

    Sets the currently active value of the TFONT attribute. Texts created after setting TFONT are displayed with the new character font. TFONT 0 and TFONT 1 are included in the system. Additional fonts can be created using the VARKON font editor.

    Available in

    2D only.

    MBS Correspondence

    set(TFONT=value);

    Comments

    TFONT can have any positive integer value, 0 included. Default value for TFONT when a new job is started is 0.
