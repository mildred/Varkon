VARKON Version 1.16 1998-12-16

f200 Dynamic rendering

Description

Displays the model in the "Dynamic Rendering" window. Using buttons in the toolbar you can switch between different rendering options that you control with the mouse. Rotation is default. By moving the mouse the model will rotate around it's centre in a direction equal to the mouse movment. Pan or Zoom or Light (Unix) or Perspective (Unix) is controlled by moving the mouse up or down in the window.

The poly button (Unix) toggles the display of the model between polygon (wireframe) mode and filled shaded mode.

The Windows95/NT version of the system is slightly different. Light, Perspective and Poly/Fill is available in the view and light dialogs.

Available in

2D and 3D.

MBS Correspondence

None.

Comments

Function f98 "Set active curve accuracy" can be used to control the rendering quality of surfaces.
