Set active attributes

All entities in Varkon have graphical attributes. For a complete list of all graphical attributes see the set() procedure. When you create an entity it automatically inherits the currently active attribute values that apply to it. The set active attributes dialog can be used to set the current value of all graphical attributes except PEN, LEVEL and WIDTH which are available in the menu window.

Select the entity type from the tabs in the top of the dialog and then edit the settings for this entity type. Next time you create an entity of this type it will get the new attribute value. Existing entities are not affected. To edit the attributes of an existing entity use menu/edit.
