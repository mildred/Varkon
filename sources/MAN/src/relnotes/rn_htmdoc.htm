Release notes for the VARKON documentation

All VARKON documentation has now been converted to HTML-format and many new pages for interactive functions have been added. Previous versions of the documentation in Microsoft Word-format or plaint text format are no longer available.

From now on there is only one version of the documentation, in HTML format, the same for all platforms and in english language. You can browse the doc-pages at our website or you can download the files and browse hem locally on your own machine. See the download pages for more info.
