Release notes for VARKON version 1.14E

The first version of VARKON available on Internet was 1.14C released in August-96 on a server owned by Signum AB in Linkoping, Sweden.

The response was very encouraging so we decided to increase our activity on Internet with a more extensive homepage, demo- versions of VARKON for Windows95/NT, demo applications and so on. At the same time we decided to move our site from Signum to a place of our own, http://www.microform.se

The first step is todays release of VARKON 1.14E for Linux together with a first release of the reference documentation for VARKON and MBS. See the README file for instructions to download the files.

Version 1.14E includes many minor enhancements and some bugfixes compared to 1.14C. An important difference is that we now have changed the names of the two main executable files included in the distribution. xv3 is now called xvarkon and v3c is now called mbsc. All scripts included in the distribution use the new names but if you have any old scripts referring to xv3 or v3c you must change these accordingly.
