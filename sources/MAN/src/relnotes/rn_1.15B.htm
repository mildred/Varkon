Release notes for VARKON version 1.15B

  • New GM-get routine GETFACP() added to MBS. GETFACP() reads patch data (polygon vertices) for faceted surfaces from GM one patch at the time.
  • New DXF-import macros. DXF-import now supports 2D as well as limited 3D import. Supported entities are POINT, LINE, ARC, TEXT, POLYLINE, SOLID and 3DFACE. Block definition and INSERT is also supported.
  • New DXF-3D export function. Converts B_planes and faceted surfaces to 3DFACE and writes to disc in DXF-format. There are many programs that can use this format. One that might be of interest is the 3D-Studio rendering program.
